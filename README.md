# Grades API

#### Grades API provides API to make store, search, delete operations easily.


## Techs
- Spring Boot 3
- Java 17
- Maven
- Spring Security
- Spring Validation
- Flyway
- PostgreSQL
- Lombok
- OpenAPI Generator
- QueryDSL
